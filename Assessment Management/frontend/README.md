# AI Internship Management System — Admin Module

Premium React frontend for the Admin & System Management module only. It covers dashboard metrics, mentor administration, student oversight, analytics, notifications, audit logs, settings, and admin profile management.

## Run

```bash
npm install
npm run dev
```

The application starts at `http://localhost:5173` and uses a session-based mock API. API service modules are isolated under `src/services/api` for a future FastAPI transition. Set `VITE_API_BASE_URL` when introducing live endpoints.
